# Sales Data Analysis Dashboard

## Project Description

This project analyzes sales data using Python, Pandas, NumPy, and Matplotlib. It loads sales records from a CSV file, cleans the data, performs analysis, generates reports, and creates visualizations.

## What I Learned

1. Data analysis using Pandas DataFrames
2. Loading and processing CSV files
3. Data cleaning and preprocessing
4. Statistical analysis of business data
5. Creating visualizations using Matplotlib
6. Exporting reports to Excel
7. Organizing Python projects into modules

## Features

* Load sales data from CSV
* Clean and preprocess data
* Calculate total sales and average sales
* Analyze sales by category
* Find top-performing categories
* Generate charts and reports
* Export data to Excel
* Modular project structure

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## Project Structure

* data_loader.py – Loads sales data
* data_cleaner.py – Cleans dataset
* analyzer.py – Performs analysis
* visualizer.py – Creates charts
* reporter.py – Exports reports
* main.py – Runs the project

## Sample Output

* Total Sales
* Average Sales
* Unique Customers
* Category-wise Sales
* Excel Report
* Sales Chart
